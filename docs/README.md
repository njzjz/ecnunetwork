**Author**: Jinzhe Zeng

**Email**: jzzeng@stu.ecnu.edu.cn

## Installation

Although you cannot connect to IPv4 servers before logining, IPv6 servers are still available, so you can use pip to install.

```sh
pip install ecnunetwork
```

## Usage
```sh
network
```
